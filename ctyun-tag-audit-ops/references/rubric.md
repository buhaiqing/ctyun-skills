# GCL Rubric

> Version: v1

## Scoring Dimensions

| # | Dimension | Scale | Threshold | Description |
|---|---|---|---|---|
| 1 | **Correctness** | 0 / 0.5 / 1 | ≥ 0.5 | Data correctly queried and parsed |
| 2 | **Safety** | 0 / 1 | = 1 | Read-only; no modifications |
| 3 | **Idempotency** | 0 / 0.5 / 1 | ≥ 0.5 | Repeated queries consistent |
| 4 | **Traceability** | 0 / 0.5 / 1 | ≥ 0.5 | Query params and response captured |
| 5 | **Spec Compliance** | 0 / 0.5 / 1 | ≥ 0.5 | Follows documented patterns |
